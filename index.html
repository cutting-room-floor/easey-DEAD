Easey is a simple map animation library. | API Docs on the Wiki | Clone & Get on GitHub Tests | Modest Maps

Easey is a map animation library. It works with Modest Maps and provides advanced easing as well as inertial panning and zooming.



easey().map(map)
  .to(map.locationCoordinate({ lat: 38, lon: -80 })
    .zoomTo(5)).run(2000);



easey().map(map)
  .to(map.locationCoordinate({ lat: 38, lon: 80 })
    .zoomTo(10))
    .path('about')
    .run(2000);



easey().map(map)
  .to(map.locationCoordinate({ lat: 60, lon: 11}))
  .zoom(8).optimal();
